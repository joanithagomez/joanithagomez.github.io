Vault Application

A password generator built using ReactJS.

Vault is an easy-to-use password generator inspired from James Coglan's Vault ( https://getvau.lt/ ). I implemented a simplified version that is responsive and easy-to-use for mobile users.

The app is built with the goal of discouraging users from reusing same passwords for different sites. This app lets users have one master password and use the service name they are logging into to generate a unique password for that service. Next time you need to log in, come back here to get your password!

Plant Identifier

A mobile application to identify plants.

The goal of this project was to build a mobile application that will instantly identify plants. We wanted to give our users an easy way to learn about plants and be more appreciative of the world around us. This app allows users to take a picture and have it classified. After classifying, users have access to information about the plant and where to buy it. Through this application, our users can have fun classifying plants either by playing a guessing mini-game after every plant they identify or playing competitively with other users.
